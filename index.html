    Сайт доступен по адресу: khabjob.ru
    Исходный код проекта: github
    Стек:

    • Aiohttp
    • PostgreSQl

    Для сбора вакансий из открытых источников используются парсеры на основе клиента библиотеки aiohttp. Проект состоит из двух частей: бекенда в виде REST API на фреймворке aiohttp и фронтенда на Vue.js. У сайта имеется закрытая часть для администраторов. Для входа в админку я решил написать собственную (без подключения готовых сторонних модулей) систему аутентификации на основе Json Web Token (JWT). Реализован полнотекстовый поиск средствами PostgreSQL по базе вакансий. Сайт развернут на виртуальном выделенном сервере (VPS) с помощью nginx и aiohttp server.

  • Городской блог Комсомольска-на-Амуре

    Сайт доступен по адресу: kmsonline.ru
    Стек:
    • Python 3.8
    • Django
    • PostgreSQL
    • Redis
    • Docker
    • Javascript
    Проект выполнен на фреймворке Django. Так как функционал сайта предусматривает включение блока рекоммендуемых статей практически на всех страницах, то было реализовано кеширование рекоммендуемых статей с помощью Redis для повышения скорости загрузки сайта. Согласно техническому заданию заказчика сайт использует систему дружественных push-уведомлений (предложение подписаться появляется не сразу, а только при достижении конца статьи). Выполнен деплой проекта на виртуальном выделенном сервере (VPS) с помощью nginx и docker.
